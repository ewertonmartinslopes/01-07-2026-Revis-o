# 📘 Revisão de Lógica de Programação em Python

Este repositório reúne uma série de exercícios de revisão desenvolvidos no **Visual Studio Code (VS Code)** utilizando a linguagem **Python**. Além da implementação dos códigos, cada atividade foi planejada por meio de um **fluxograma criado no Draw.io (diagrams.net)**, permitindo visualizar toda a lógica antes da programação.

O objetivo deste projeto é consolidar os conceitos fundamentais de lógica de programação, praticando desde operações matemáticas simples até estruturas condicionais e de repetição, seguindo boas práticas de desenvolvimento.

## 🎯 Objetivos do Projeto

* Desenvolver algoritmos utilizando Python.
* Exercitar o raciocínio lógico.
* Planejar soluções por meio de fluxogramas.
* Aplicar conceitos de programação estruturada.
* Organizar projetos utilizando o GitHub como portfólio.

---

## 📚 Exercícios Desenvolvidos

### 🔹 Questão 1 – Dobro de um Número

Desenvolvimento de um algoritmo que solicita um número inteiro ao usuário e exibe o dobro do valor informado.

**Conceitos praticados:**

* Entrada de dados (`input`)
* Conversão de tipos (`int`)
* Operadores aritméticos
* Saída de dados (`print`)

---

### 🔹 Questão 2 – Verificando a Idade

Programa que lê a idade do usuário e informa se ele é maior ou menor de idade utilizando estruturas condicionais.

**Conceitos praticados:**

* Estruturas condicionais (`if` e `else`)
* Operadores relacionais
* Tomada de decisão

---

### 🔹 Questão 3 – Média de Notas

Programa responsável por receber duas notas, calcular a média aritmética e apresentar o resultado ao usuário.

**Conceitos praticados:**

* Variáveis
* Operadores matemáticos
* Conversão de dados (`float`)
* Cálculo da média

---

### 🔹 Questão 4 – Contagem Crescente

Programa que utiliza uma estrutura de repetição para exibir os números de **1 até 10**.

**Conceitos praticados:**

* Estrutura de repetição (`for`)
* Função `range()`
* Controle de laços

---

### 🔹 Questão 5 – Calculadora Simples

Programa que solicita dois números ao usuário e realiza as quatro operações matemáticas básicas:

* Soma
* Subtração
* Multiplicação
* Divisão

Os resultados são apresentados de forma organizada na tela.

**Conceitos praticados:**

* Operadores aritméticos
* Entrada e saída de dados
* Variáveis
* Cálculos matemáticos

---

## 📊 Fluxogramas

Antes da implementação de cada exercício, foi desenvolvido um fluxograma utilizando o **Draw.io (diagrams.net)** para representar visualmente a lógica dos algoritmos.

Os fluxogramas permitem:

* Compreender o fluxo de execução do programa.
* Planejar a solução antes da codificação.
* Facilitar a identificação de decisões e repetições.
* Desenvolver boas práticas na construção de algoritmos.
